# EcoMonitor 🌍💨

**EcoMonitor** — це веб-застосунок для автоматизованого моніторингу, аналізу та динамічної візуалізації екологічних показників якості атмосферного повітря в реальному часі. Проєкт розроблено в межах курсової роботи з дисципліни «Об'єктно-орієнтоване програмування» за спеціальністю 121 «Інженерія програмного забезпечення».

---

## 🚀 Функціональні можливості

* **Моніторинг показників у реальному часі:** Відображення концентрацій твердих частинок (PM2.5, PM10), діоксиду азоту (NO₂) та базових метеорологічних даних (температура, вологість).
* **Автоматичний розрахунок статусів (AQI):** Алгоритми обробки даних динамічно визначають рівень екологічної безпеки та застосовують відповідну колірну індикацію за стандартами ВООЗ.
* **Динамічна візуалізація часових рядів:** Генерація графіків коливань показників на серверній стороні (Server-Side Rendering) на основі останніх 20 вимірювань за допомогою бібліотеки `JFreeChart`.
* **Інтерактивні медичні поради:** Спливаючі модальні вікна Bootstrap з екологічними довідками та порадами для населення залежно від поточного стану повітря.
* **Історія вимірювань:** Збереження та відображення журналу останніх записів у вигляді інтерактивної таблиці з фільтрацією.

---

## 🛠 Технологічний стек

* **Мова програмування:** Java 17 (LTS)
* **Фреймворк:** Spring Boot 3.x (Spring Web, Spring Data JPA)
* **Шаблонізатор (View):** Thymeleaf + HTML5 / CSS3 / JavaScript
* **Стилізація інтерфейсу:** Bootstrap 5
* **База даних:** PostgreSQL
* **Об'єктно-реляційне відображення (ORM):** Hibernate
* **Візуалізація даних:** JFreeChart 1.5.4
* **Збирання проєкту:** Maven
